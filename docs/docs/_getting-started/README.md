## ⚡ Quickstart

[👉 Ver Quickstart](./quickstart.md)

**Em 5 minutos você vai ter a aplicação rodando.**

Pré-requisitos:
- Node.js 18+
- PostgreSQL 14+
- Git

Resumo dos passos:
1. Clone o repo
2. `npm install`
3. Configure `.env`
4. Crie o banco: `createdb abp`
5. `npm run db:init`
6. `npm run dev`
7. Abra http://localhost:3005
