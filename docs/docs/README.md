## 🔄 Mantendo a Documentação Atualizada

Se você vai adicionar um novo documento:

1. **Identifique a categoria** — em qual pasta ele se encaixa?
2. **Coloque na pasta correspondente** — crie uma nova pasta se necessário
3. **Atualize o `README.md` da pasta** — adicione um link para o novo doc
4. **Faça um commit** — `docs: adiciona <documento> em <pasta>`

Exemplo:
```bash
# Novo doc sobre autenticação
docs/_development/authentication.md

# Atualize:
docs/_development/README.md
```
